수도권 1km 격자 인구밀도 지도

통계청 국토통계지도 1km 격자 인구(2024.10)를 기반으로, 서울·인천·경기 13,052개 격자의 인구밀도(명/㎢)를 보여줍니다. 색상이 진할수록 인구밀도가 높으며, 등간격 7단계로 분류하고 최상위 구간은 상한 없이 ‘이상’으로 열어두었습니다.

사용법
  • 격자에 마우스 올림/탭 → 시도·시군구·읍면동·인구·밀도
  • 범례 구간 클릭 → 해당 밀도 구간 격자만 강조(재클릭 해제)
  • 좌상단 검색 → 읍면동으로 이동·강조
  • 밀도 범위 슬라이더 → 특정 밀도 범위만 표시
  • 좌하단 레이어 → 레이어 켜고 끄기
